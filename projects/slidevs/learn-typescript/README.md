# learn-typescript

20220906 typescript技术分享

使用slidev技术进行内容演示，其中`slides.md`,`ts-classes.md`,`ts-declaration.md`是主要文档  
slides-export文件夹内存放导出的pdf文件  
其中src文件下存放示例代码，按内容演示的顺序标好了数字  
type文件夹存放示例的声明文件


## 使用方法

### 安装项目依赖

```shell
yarn
```
### 运行项目
```shell
yarn start
```

### 导出项目pdf
```shell
yarn slidev:export
```
