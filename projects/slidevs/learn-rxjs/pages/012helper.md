## 辅助类操作符

| 功能需求                                 | 适用的操作符    |
| ---------------------------------------- | --------------- |
| 统计数据数据流产生的所有数据个数         | count           |
| 获得数据流中最大或者最小的数据           | max和min        |
| 对数据流中所有数据进行reduce操作         | reduce          |
| 判断是否所有数据满足某个判定条件         | every           |
| 找到第一个满足判定条件的数据             | find和findIndex |
| 判断一个数据流是否不包含任何数据         | isEmpty         |
| 如果一个数据流为空就默认产生一个指定数据 | defaultEmpty    |
